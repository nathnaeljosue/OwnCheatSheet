COMMON HTML AND CSS PROPERTIES

HTML PROPERTIES

Tag Name Description
<div> Division Defines a section in a document.
<span> Span Groups inline elements for styling.
<a> Anchor Defines a hyperlink.
<img> Image Embeds an image.
<ul> Unordered List Defines a bulleted list.

CSS PROPERTIES

Tag Name Description
color Text Color Specifies the color of text.
background-color Background Color Sets the background color.
margin Margin Sets the outer spacing.
padding Padding Sets the inner spacing.
font-size Font Size Specifies the size of text.
